# Pilothouse UI kit
Frostyard-consistent restyle of Pilothouse (https://github.com/frostyard/pilothouse), the local web admin console for Snosi installations. The layout, class structure, and module inventory mirror the real app (`internal/web/static/app.css`: app-shell, sidebar, dashboard-grid, metric cards, data tables, login split) — the light sage/lime theme is replaced with the frostyard-org language: ink surfaces, hairlines, ice/sky accents, square corners, mono kickers. State colors (--state-ok/warn/danger) are intentional token additions for admin surfaces. Screens: Login, Overview, Sysexts, Services, Podman.
